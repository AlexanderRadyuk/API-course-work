Для работы приложения необходимо заплолнить данные в файле "input_data.py", в который внести следующие данные в следующем формате:

app_id: 'идентификационный номер приложения',
user_id: 'номер пользователя ВКонтакте',
ТОкен Вконтакте.
Для получения токена ВКонтакте создать экземпляр класса VKtoken, применить метод app_auth к созданному экземпляру. В полученном сообщении скопировать
адрес и ввести его в адресную строку браузера. Перейти по адресу и в полученном адресе скопировать токен и ввести его в файл "input_data.py"
VK_token: 'vk1.a.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
Для получения токена Яндекс войти в аккаунт, перейти в API Яндекс Диска, затем перейти в Полигон. На странице скопировать токен и ввести его
в файл "input_data.py" в раздел Yandex_token, сохранив заголовок OAuth, через пробел"
Yandex_token: 'OAuth xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'



Для выгрузки фотографий из страницы ВКонтакте создать экземпляр класса VKphoto  и испольвать метод get_vk_photo
Для создания списка файлов фотографий на загрузку создать экземпляр класса FileListCreate испольвать метод listcompile
Для создания папки в Яндекс Диске для загрузки создать экземпляр класса YandexFolder и использовать метод foldercreate
Для загрузки фото в папку Яндекс Диска создать экземпляр класса YandexUpload и использовать метод load_files.
